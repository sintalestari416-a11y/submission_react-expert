# Forum Diskusi - React Expert Submission

Aplikasi forum diskusi berbasis React menggunakan Redux Toolkit, React Router, dan Vitest.

## Fitur
- Autentikasi pengguna (Login & Registrasi)
- Daftar thread diskusi
- Buat thread baru
- Komentar pada thread
- Leaderboard pengguna aktif

## Teknologi
- React 18 + Vite
- Redux Toolkit
- React Router v6
- Vitest + Testing Library
- Cypress (E2E)
- ESLint (Airbnb config)

- Simulasi Pull Request untuk Branch Protection
